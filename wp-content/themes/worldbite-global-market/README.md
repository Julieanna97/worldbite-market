# WorldBite Global Market

A classic WordPress/WooCommerce theme based on the approved clean global-food marketplace design.

## Main design features
- Dark-green benefit strip
- Logo, full product search, account and cart header
- Cuisine navigation bar
- Large editorial food hero
- Circular cuisine browsing
- Six-column featured product grid
- Global collections and collection-builder styling
- Responsive shop, product, cart, checkout and account pages
- Built-in image fallbacks for the demo products

## Install
1. In WordPress go to **Appearance → Themes → Add New Theme → Upload Theme**.
2. Upload `worldbite-global-market.zip`.
3. Install and activate it.
4. Go to **Settings → Permalinks** and click **Save Changes** once.
5. Clear your browser cache and reload the homepage.
