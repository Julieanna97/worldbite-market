**Setup**
1. Activate this plugin.
2. Create a page "Create "Create Collection" with content: `[mp_create_collection]`.
3. Add WooCommerce products.
4. Add terms under **Collections -> Collection Categories**.
5. Visit `/collections (achive)` and the Create page to test.
6. (Analytics) GTM container should include a GA4 config (All Pages) and a GA4 Event tag for `collection_created`.
